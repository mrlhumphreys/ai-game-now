# AI Game Now

Play board games against an AI.

Frontend written in Svelte. 

## URL

https://mrlhumphreys.github.io/ai-game-now/

## Games Available

* Checkers
* Backgammon

## Environment Variables 

* `AI_SERVICE_URL` - The url of the api service .

## How to run in dev

```bash
  yarn run dev
```
